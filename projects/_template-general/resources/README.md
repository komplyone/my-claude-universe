# Resources

This folder contains information context for the project.

---

## How to Use

Add files here that provide context for the project:

- **Research documents** - Studies, reports, analysis
- **Reference materials** - External docs, standards, guidelines
- **Meeting notes** - Summaries of discussions and decisions
- **Links and summaries** - Curated external resources
- **Any information Claude should know about**

### File Naming

Use descriptive names with dates where relevant:
- `2024-01-market-research.md`
- `competitor-analysis.md`
- `meeting-notes-kickoff.md`
- `external-api-docs.md`
