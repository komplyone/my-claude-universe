## Decisions

### DEC-2025-001: Astro for Static Site
**Date:** 2025-01-15
**Context:** Selecting framework for corporate website
**Decision:** Use Astro with Tailwind CSS
**Rationale:**
- Static-first is appropriate for marketing site
- Consistent with Recoger website approach
- Fast, SEO-friendly output
- Simple deployment to Cloudflare Pages
**Alternatives Considered:**
- Next.js: Overkill for static content
- Plain HTML: Less maintainable
**Reversible:** Yes
