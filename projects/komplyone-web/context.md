# KomplyOne Website — Project Context

> Corporate website for the KomplyOne parent brand.

---

## Overview

The KomplyOne website (komply.one) serves as the corporate home for the brand family. It showcases consulting services and links to product offerings (Recoger, Velador).

---

## Current State

**Status:** Planned

Lower priority than product-specific websites. Will be developed after Recoger website is complete.

---

## Purpose

### Primary Goals
1. Establish KomplyOne as the parent brand
2. Showcase ISO 27001 consulting services
3. Link to product portfolio (Recoger, Velador)
4. Provide company information and contact

### Target Audience
- Potential consulting clients
- Enterprise procurement researching vendor
- Investors and partners
- Job seekers

---

## Site Structure

```
komply.one/
├── /                    # Homepage
├── /products            # Product portfolio
│   ├── /recoger        # Recoger overview → recoger.co
│   └── /velador        # Velador overview → velador.app
├── /consulting          # ISO 27001 services
├── /about               # Company story
├── /contact             # Contact information
└── /legal
    ├── /privacy
    └── /terms
```

---

## Design Direction

- Professional, trustworthy aesthetic
- Nordic-inspired clean design
- Dark mode compatible
- Consistent with product brand family
- Mobile-first responsive
