### Assignees
- `@user` - Human is responsible
- `@claude` - Claude handles autonomously
- `@together` - Collaborative effort

### Priority Tags
- `#critical` - Drop everything
- `#high` - Do soon
- `#normal` - Standard (default)
- `#low` - When time permits

### Adding Tasks

- "Add user authentication @claude #high"
- "Review API design @user"
- "Implement caching @together"

- "Done with [task]"

- "Mark [task] as complete"
